Commands To Navigate The Filesystem

How do I get the path to where I currently am in the directory? (pwd)

pwd prints the path of your current/working directory.

$ pwd /home/user/workspace-cs240
How do I change directories? (cd)

cd command lets you move into another folder.

$ cd Documents

Another example:

$ cd workspace-cs240/Hw1

To move back one directory:

$ cd ..
How do I list the contents of a directory? (ls)

ls lists directory contents

$ ls Q1.c  Q2.c  Q3.c  makefile  readme $

Commands To Create and Delete Files and Directories

How do I create a new directory? (mkdir)

Type mkdir followed by the desired directory name.

For example, to create a directory to store all of your CS240 homework assignments:

$ cd $HOME $ mkdir workspace-cs240 $ cd workspace-cs240 $ mkdir Hw1
How do I create a new file? (touch)

Use touch followed by a filename to create an empty file.

$ touch Q1.c
How do I delete files and directories (rm & rmdir)

To delete a directory that is empty:

$ rmdir Hw0

To permanently delete a file:

$ rm hello.c

Compiling your C pset

How do I compile my C program? (make)

Open your terminal and navigate to your homework directory (e.g., home/user/workspace-cs240/Hw1). Then type make to compile the code based on the instructions in your makefile. This will generate the executable files.

$ ls makefile readme Q1.c Q2.c Q3.c $ make $ ls makefile readme Q1 Q1.c Q2 Q2.c Q3 Q3.c
Now, run the executable:
$ ./Q1
To remove the executables:
$ make clean $ ls $ makefile readme Q1.c Q2.c Q3.c

Submitting Homework to the UMB CS Server

How do I copy my files to the CS server? (scp)

Open your terminal and navigate to the directory where your homework is saved. (e.g., /home/users/workspace-cs240/Hw1)

To copy a file to the UMB CS server:

$ scp Q1.c username@users.cs.umb.edu:/home/username/cs240/Hw1

Another example:

$ scp makefile username@users.cs.umb.edu:/home/username/cs240/Hw1

username should be replaced with your actual UMass Boston CS account username.

After copying your files, log in to the CS server via SSH to confirm they were copied successfully.


Note: Submissions must include both a makefile and a readme to receive full credit.

Reminder: Be sure to review the official UMB_Unix_Server_Tutorial (PDF) provided by your instructor for full instructions.

How do I log in to the CS server via SSH?

Use the following command in your terminal:

$ ssh username@users.cs.umb.edu

username should be replaced with your actual UMass Boston CS account username.

How do I set permissions on my submitted files?

From your homework directory on the CS server, run the following commands for each file you submitted:

username@pe15$ chmod o-r Q1.c
username@pe15$ chmod g+r Q1.c

Repeat this for all submitted files, including your makefile and readme
